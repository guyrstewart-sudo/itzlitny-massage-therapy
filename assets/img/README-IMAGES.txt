SWAP IN BRITTANY'S REAL PHOTOS — no code changes needed
=========================================================
Drop image files into THIS folder (assets/img/) using these exact names.
They auto-replace the styled placeholders the moment they exist.

  portrait.jpg    → About section portrait (4:5 vertical, warm low light)
  room.jpg        → Ritual: treatment room / warm linens / candlelight (4:5)
  detail.jpg      → Ritual: hands / warm stones / oil (16:9 wide)
  gallery-1.jpg   → Cascading gallery (portrait works best)
  gallery-2.jpg   → Cascading gallery
  gallery-3.jpg   → Cascading gallery
  gallery-4.jpg   → Cascading gallery
  gallery-5.jpg   → Cascading gallery
  gallery-6.jpg   → Cascading gallery
  space.jpg       → Contact: photo of the studio or a map screenshot (4:3)

DARK THEME: photos with warm, low-light, moody tones suit the new design best.
The gallery is a cascading/parallax layout — vertical (portrait) shots look best.

TIPS
- Recommended widths: portrait ~1000px, room/detail ~1600px, gallery ~1200px.
- Keep files under ~400 KB each (compress at tinypng.com) so the site stays fast.
- .jpg names are referenced above; if you prefer .png/.webp, edit IMG_MAP in
  assets/js/main.js to match your filenames.
